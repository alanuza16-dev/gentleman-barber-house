# The Gentleman Barber House

Sitio base para una barberia clasica con estetica elegante, ambientacion tipo gentleman y estructura derivada de `lanzah03`.

## Incluye

- hero con imagenes de barberia clasica
- seccion de identidad de marca
- catalogo de servicios con modales
- contacto y CTA por WhatsApp
- branding inicial, listo para personalizar
